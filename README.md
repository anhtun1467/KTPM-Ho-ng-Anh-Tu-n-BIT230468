## Mô tả
Repository này chứa toàn bộ các bài thực hành kiểm thử phần mềm
theo phần "Bài đọc thêm" do các trợ giảng AI hướng dẫn.

## Danh sách bài thực hành
1. Unit Testing với JUnit
2. API Testing với Postman
3. UI Testing với Cypress/Playwright
4. Performance Testing với JMeter/k6
5. Security Testing với OWASP ZAP
6. Static Code Analysis với SonarQube
7. Bug Management với GitHub Issues/Jira
8. Kiểm thử với công cụ sử dụng LLM
